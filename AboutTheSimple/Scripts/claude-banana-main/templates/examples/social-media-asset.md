# Social Media Asset

## Domain Mode
Digital Marketing

## Technique
Social Media Frame Integration

## Prompt Template

A creative illustration showing {character} interacting physically with the user interface elements of a {platform} post layout. The screen displays a realistic recreation of the platform's UI, including the profile picture, the username "{username}" in the correct font and placement, and standard buttons like like, comment, and share. The character reaches out from within the post image to {interaction}, treating the flat UI elements as tangible three-dimensional objects. The character's body exists partly inside the post's image area and partly overlapping the UI chrome, breaking the expected visual hierarchy. The overall color scheme matches the platform's brand palette, and the typography is clean and contemporary. The rendering style blends the illustrated character seamlessly with the flat UI, as though the two visual languages have merged. A subtle drop shadow appears where the character's hands or body overlap the interface elements, reinforcing the illusion of dimensionality.

## Variables

- `{character}`: The illustrated subject — e.g., "a cheerful cartoon fox mascot in a chef's hat," "a young woman in a leather jacket with headphones around her neck"
- `{platform}`: Social media platform whose UI is recreated — e.g., "Instagram," "TikTok," "Twitter/X," "YouTube"
- `{username}`: Display name shown in the post — e.g., "@foxykitchen," "@beatsby_nova"
- `{interaction}`: How the character engages with the UI — e.g., "grab the like button heart and hold it up triumphantly," "sit on top of the comment box with legs dangling," "peel back the corner of the post image like a sticker"

## Aspect Ratio
4:5

## Usage Notes
This template is perfect for social media managers, brand campaigns, and content creators who want eye-catching posts that play with the platform's own visual language. Nano Banana Pro (gemini-3.1-flash-image-preview) handles the blend of illustrated characters and flat UI elements convincingly. Ensure the platform UI elements are recognizable but avoid using exact trademarked logos — approximate the layout instead.

## Example Output

A creative illustration showing a cheerful cartoon fox mascot wearing a tall white chef's hat and a red apron interacting physically with the user interface elements of an Instagram post layout. The screen displays a realistic recreation of Instagram's UI, including a round profile picture of the fox, the username "@foxykitchen" in the correct font and placement, and standard buttons like the heart, comment bubble, and paper airplane share icon. The fox reaches out from within the post's image area to grab the like button heart and hold it up triumphantly above its head, with the heart glowing pink and enlarged in its paw, treating the flat UI element as a tangible three-dimensional object. The fox's body exists partly inside the post's image area — a cozy kitchen scene — and partly overlapping the UI chrome below, breaking the expected visual hierarchy. The overall color scheme matches Instagram's white, gray, and soft gradient brand palette, and the typography is clean and contemporary. The rendering style blends the illustrated fox seamlessly with the flat UI, as though the two visual languages have merged. A subtle drop shadow appears where the fox's paws overlap the interface elements, reinforcing the illusion of dimensionality.
