# UI Mockup

## Domain Mode
UI/Web

## Technique
Glassmorphism with layered translucency and depth hierarchy

## Prompt Template

A polished UI mockup for {app_type} displayed on {device}, rendered in a {style_direction} design language. The interface employs glassmorphism as its primary visual motif: frosted translucent panels float above a softly blurred background, with subtle white borders catching simulated light along their top and left edges. The color scheme is built on {color_scheme}, applied consistently across interactive elements, typography hierarchy, and data visualization components. Cards and containers use varying levels of backdrop blur — heavier blur for primary content, lighter blur for secondary panels — creating a natural depth hierarchy without relying on drop shadows alone. Typography is set in a geometric sans-serif, with clear size and weight distinctions between headings, body text, and labels. Interactive elements such as buttons and toggles feature micro-interaction states visible in the design: hover glows, pressed states, and focus rings. The layout follows a twelve-column grid with generous padding, and the overall composition includes enough realistic placeholder content — names, numbers, avatars, and chart data — to feel like a functioning application rather than an empty wireframe.

## Variables

- `{app_type}`: The application category — e.g., "a personal finance dashboard," "a music streaming player," "a fitness tracking app," "a project management tool"
- `{color_scheme}`: The color foundation — e.g., "deep indigo and electric violet with white text," "muted sage green and warm cream," "dark mode with charcoal backgrounds and neon cyan accents," "warm gradients from coral to amber"
- `{device}`: The display context — e.g., "an iPhone 15 Pro," "a MacBook Pro 16-inch screen," "an iPad in landscape orientation," "a 27-inch desktop monitor"
- `{style_direction}`: The design personality — e.g., "clean and corporate," "playful and rounded," "dark and futuristic," "warm and organic"

## Aspect Ratio
Device-dependent (9:19.5 for mobile, 16:10 for laptop, 4:3 for tablet)

## Usage Notes
UI mockups generated from prompts are best used for concept exploration, pitch decks, and design direction — not production assets. The glassmorphism technique works reliably with Nano Banana Pro (gemini-3.1-flash-image-preview), which understands translucency and interface conventions well. Including specific content placeholders in the prompt dramatically improves realism. For mobile screens, use portrait aspect ratios; for desktop, use landscape.
